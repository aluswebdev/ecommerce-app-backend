# E-commerce Mobile App - Backend

This is the **backend** of the mobile e-commerce app, built with **Node.js / Express**, serving users in [Sierra Leone].  

The backend is **private** and only meant for collaborators who are approved.

## Purpose
- Handle user authentication, product management, cart and checkout operations.
- Provide secure API endpoints for the frontend.
- Allow contributors to extend features for local use.

## Tech Stack
- Node.js / Express
- MongoDB / Mongoose (or your database)
- JWT for authentication
- dotenv for environment variables

## For Contributors
- Contact the project owner before cloning the repository.
- Use the development branch for contributions.
- Never commit `.env` or other sensitive files.
- All contributions go through pull requests after approval.

## Notes
- This repo is **not intended for public use**, only for approved contributors.
